Countdown Plugin Description

The Coundown plugin above is called dsCountDown.It counts down to any date you set.
First you need to download the plugin from the following link.

Download Link

The following are the steps to installing this plugin.

1. Include jQuery javascript library and jQuery dscountdown plugin in the page.

2. Create a container for the countdown timer.

3. Load the required stylesheet file for the basic style and theme styles.

4. Call the plugin and set the end date you want to count down to in the javascript.

Whats On And Where Is It?

Big Names

EGX hosts hundreds of games each year for players to get an early look at upcoming releases. Some of the worlds top Developers will be in attendance with some offering interviews and dev talks. The biggest gaming companies will have their own section housing all their own IPs and more

Indie Support

There is a huge amount of Indie games coming this year also and the creators will be right there to show you the ropes of their game. EGX is a great place to show off your skills as a gamer but also as a developer as your main audience will all be in the one place

Venue

EGX 2019 will be held at ExCeL London in the Docklands. The address is ExCeL London, One Western Gateway, Royal Victoria Dock, London E16 1XL. 80,000 people attended EGX in 2017 for the 10th Anniversary. Lets try break that record this year
